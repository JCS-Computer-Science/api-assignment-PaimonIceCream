This main purpose of this site is to fetch data from an API through a URL. If you were to find a website containing API data, the data would be acconpanied by a URL and would be stored as JSON data that would look similar to the example below.

{
      "key": "value",
      "boolean": "true",
      "number": 10
}

Once you have obtained a copied valid URL, try it out on the API fetcher by pasting it into the white bar that says Fetch API URL Here and clicking the Submit button. A valid URL would show the prompt Please Select a Key beside where you pasted the URL. That prompt would open a dropdown containing some keys. Each key corresponds to its respective value that will be seen in the box highlighted green at the very bottom of the API fetcher.

You may also choose to store any URLs you come across on the API fetcher for later use. Once you have copied a URL, paste it in the box with the prompt Store API URL Here and click the Store API button to store it in the list below. The URL will show in the list and can be selectable for copying and pasting for fetching data. If you wish to remove a URL from the list, you can double click the URL to clear it.

The first two URLs in the list are examples that you can try out and cannot be removed from the list. Have fun!
